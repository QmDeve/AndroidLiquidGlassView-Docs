## Integration

### Add Dependencies <Badge type="tip" text="v1.0.2" />
In your module's `build.gradle` file, add the dependency:
::: code-group
```groovy [build.gradle]
dependencies {
   implementation 'com.qmdeve.liquidglass:core:1.0.2'
}
```

```kotlin [build.gradle.kts]
dependencies {
   implementation("com.qmdeve.liquidglass:core:1.0.2")
}
```
:::
